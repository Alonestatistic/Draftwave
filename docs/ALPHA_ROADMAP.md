# Draftwave Private Alpha Roadmap

## Product Identity

The first shippable identity is Creative Sketch DAW, not a pro DAW competitor. The release promise is fast ideation, reliable project persistence, basic audio, and clear capability boundaries.

## Track

- Alpha 0: stabilize the Electron/Vite shell, error surfaces, project migrations, autosave recovery, smoke checks, and release docs.
- Alpha 1: harden the core workflow for create/edit/arrange/import/record/save/load/undo/export.
- Alpha 2: polish musical tools such as piano roll editing, sampler/drum rack basics, shortcut settings, and honest mixer effects.
- Alpha 3: package Windows builds with versioning, release notes, installer metadata, and issue reporting.
- Alpha 4: run private tester feedback against known-limitations docs and data-loss stop-ship gates.

## Public Beta Gate

Public beta is blocked until save/load/export are reliable across tester machines and there are no known data-loss bugs.
