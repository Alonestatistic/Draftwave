# Private Alpha Quickstart

## Start The App

1. Run `npm install`.
2. Run `npm start`.
3. Use the desktop window, not the raw browser tab, for save dialogs, recording permissions, and export dialogs.

## Make A Sketch

1. Open the app menu and choose `New - Clean Slate`.
2. Add a Drum Rack or Built-in Sampler track.
3. Add an instrument track and open the Piano Roll.
4. Draw or edit notes, then press Space to play.
5. Use the arrangement view to move, split, duplicate, or delete clips.
6. Open the Mixer tab and adjust track volume or pan.

## Audio Workflow

1. Use `Import Audio (WAV/MP3/OGG/FLAC)` for local files.
2. Use `Microphone Recording` only on machines where the runtime grants mic access.
3. Save the project after importing or recording audio.
4. Reopen the project and confirm imported or recorded clips still appear before continuing a longer test.

## Save, Restore, Export

1. Save with `Ctrl+S` or `Save Project As...`.
2. If the app finds an autosave on launch, choose Restore or Discard before starting a new test.
3. Export with `Export Mixdown WAV`.
4. Verify the WAV opens in a normal desktop player.

## Report A Problem

Use `Report Issue...` from the app menu after a crash, device failure, save/load failure, import failure, recording failure, or export failure. The report includes app metadata, capability status, project summary, and recent error strings, but not private audio.
