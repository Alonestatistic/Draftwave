# Draftwave

Draftwave is a Windows-first creative sketch DAW for quickly turning musical ideas into arranged sessions. It combines audio and MIDI tracks, imported sounds, browser-based instruments, project save/load, export workflows, and an Assistant Producer that can use local models such as Ollama when cloud AI is unavailable.

## Current Focus

Draftwave's functional app surface is focused on a reliable creative sketch workflow:

- Create and arrange audio, MIDI, drum, bass, pad, lead, key, sampler, and drum-rack tracks.
- Upload or drag in WAV, MP3, OGG, FLAC, and WEBM sounds.
- Save, load, autosave, and restore projects with embedded media.
- Export WAV mixdowns.
- Use the Assistant Producer for quick musical actions and ideas.
- Run local AI through Ollama as a fallback or primary engine.

Prototype-only ideas that are not functional in the DAW are intentionally kept out of the active app surface and release checks.

## Development

```bash
npm install
npm run check
npm run start
```

## Build The Windows Installer

```bash
npm run package:win
```

The installer is generated by Electron Builder and copied to:

```text
release/Draftwave-Setup-0.1.0-x64.exe
```

Generated installers and build output are intentionally ignored by Git. The GitHub Actions workflow at `.github/workflows/windows-installer.yml` also builds the installer and uploads it as a workflow artifact.
